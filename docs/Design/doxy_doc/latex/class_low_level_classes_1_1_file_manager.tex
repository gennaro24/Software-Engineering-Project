Carica il file JSON presente tra le risorse del progetto e restituisce una {\ttfamily \doxylink{class_low_level_classes_1_1_contact_list}{Contact\+List}}. 

Se il file {\ttfamily contact.\+json} non viene trovato tra le risorse, viene restituita una lista vuota.
