int Low\+Level\+Classes.\+Contact.\+compare\+To (\begin{DoxyParamCaption}\item[{\mbox{\hyperlink{class_low_level_classes_1_1_contact}{Contact}}}]{o}{}\end{DoxyParamCaption})}

Confronta questo contatto con un altro contatto per determinare l\textquotesingle{}ordine alfabetico. 

La logica di confronto è la seguente\+: 
\begin{DoxyItemize}
\item Se entrambi i cognomi sono non vuoti, vengono confrontati in ordine alfabetico. 
\item Se i cognomi sono uguali, vengono confrontati i nomi. 
\item Se uno dei campi (nome o cognome) è vuoto, il contatto con il campo non vuoto viene considerato "{}minore"{}. 
\item Se il confronto tra cognomi e nomi non restituisce una differenza, i contatti sono considerati uguali. 
\end{DoxyItemize}
