static ArrayList< Contact > LowLevelClasses.ContactListManager.searchContact ( ArrayList< Contact > contacts,
String query )
static

Cerca contatti che contengono la query specificata nel nome o nel cognome.

Il metodo filtra la lista dei contatti confrontando (in modo case-insensitive) la concatenazione del nome e del cognome con la query fornita. La lista risultante viene poi ordinata in ordine naturale.

Parametri
contactsLa lista di contatti in cui effettuare la ricerca.
queryLa stringa (o sottostringa) da cercare. Se null, viene restituita la lista originale.
Restituisce
Una lista di contatti che corrispondono alla query, ordinata alfabeticamente.
